File Upload Guide

Supported File Types

  • Markdown (.md)
  • Text files (.txt)
  • PDF documents (.pdf)
  • Word documents (.doc, .docx)
  • Images (.png, .jpg, .jpeg)

Upload Process

  1. Select files using the upload button or drag and drop
  2. Files are processed and stored securely
  3. You can view and manage uploaded files in your dashboard

File Size Limits

Maximum file size: 10MB per file
